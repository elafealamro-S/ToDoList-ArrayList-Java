# Dynamic To-Do List Application

A Java console application that creates and manages a personalized To-Do list using a dynamic `ArrayList` of custom task objects.

## 🚀 How it Works
The program loops to prompt the user for task entries, creates a `Tasks` object for each, and stores them dynamically:
* **Task Title:** Name or description of the task.
* **Due Date:** Deadline for completion.
* **Status:** Numeric representation of the task progress.
